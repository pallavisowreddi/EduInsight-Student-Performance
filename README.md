# 🎓 EduInsight – Student Performance Analytics
## 📌 Introduction
EduInsight is a Machine Learning-based web application that analyzes student data and predicts whether a student is likely to pass or fail.  
It also provides personalized study recommendations to help improve academic performance.

## 🎯 Objective
The main goal of this project is to:
- Predict student performance using Machine Learning
- Identify factors affecting student success
- Provide personalized suggestions for improvement

## 🚀 Features
- 📊 Predicts student performance (Pass/Fail)
- 📈 Shows model confidence score
- 🧠 Provides personalized study advice
- 💻 Simple and interactive user interface
- ⚡ Fast prediction using trained ML model

## 🛠️ Technologies Used
- Python
- Pandas & NumPy (Data Processing)
- Scikit-learn (Machine Learning Model)
- Streamlit (Web Application)
- Joblib (Model Loading)

## 📊 Input Parameters
The system takes the following inputs:
- Age
- Gender
- Class
- Study Hours per Day
- Attendance Percentage
- Parental Education
- Internet Access
- Extracurricular Activities
- Subject Scores (Math, Science, English)
- Previous Year Score

## ⚙️ How the Project Works
1. User enters student details in the interface  
2. Data is preprocessed and converted into required format  
3. Machine Learning model analyzes the data  
4. Model predicts:
   - Pass OR Fail  
5. System generates:
   - Prediction result  
   - Confidence score  
   - Personalized study suggestions  

## ▶️ How to Run the Project
### Step 1: Install required libraries
pip install -r requirements.txt

### Step 2: Run the application
streamlit run app.py
