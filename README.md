# 🎓 EduInsight - Student Performance Analytics
## 📌 Overview
EduInsight is a Machine Learning-based web application that predicts student performance and provides personalized study recommendations based on input data.

## 🚀 Features
- Predicts whether a student will Pass or Fail  
- Provides personalized study suggestions  
- Displays model confidence score  
- Simple and user-friendly interface  

## 🛠️ Technologies Used
- Python  
- Pandas  
- NumPy  
- Scikit-learn  
- Streamlit  

## 📊 How It Works
1. User enters student details (age, study hours, attendance, etc.)  
2. Data is processed and sent to the trained ML model  
3. Model predicts performance (Pass/Fail)  
4. System generates personalized study advice
